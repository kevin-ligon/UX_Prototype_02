This package helps non-technical designers convert Figma designs into interactive HTML prototypes using Claude Code.

## Quick Start

**Start here:** Read `BOOTSTRAP.md` for complete setup and usage instructions.

## What's Included

- **3 Pre-configured Agents**: Generic Agent, Icon Maker, UI Builder
- **4 Specialized Skills**: `/publish`, `/trace-map-regions`, `/create-ui-from-image`, `/crop-icons`
- **Starter Files**: Basic HTML/CSS template with mobile-first responsive design
- **GitHub Pages Deployment**: Automated workflow for publishing your prototype

## Requirements

- Node.js 24 or higher
- Git
- Claude Code (CLI or Desktop)
- GitHub account (for deployment)

## Getting Started

1. Extract this package to your project folder
2. Open `BOOTSTRAP.md` and follow the setup instructions
3. Run `npm install` in your terminal
4. Start building from your Figma designs!

Built for UX designers who want to prototype without code knowledge.
